# CodeAlpha C Programming Internship

Welcome to my **CodeAlpha C Programming Internship** repository! This repository contains the projects and programs I completed during my internship to strengthen my understanding of the C programming language, problem-solving, and core programming concepts.

## About

This repository showcases my internship work completed as part of the **CodeAlpha C Programming Internship**. During the internship, I developed several console-based applications using fundamental C programming concepts such as:

* Variables and Data Types
* Conditional Statements
* Loops
* Functions
* Arrays
* Structures
* File Handling
* Menu-Driven Programming

These projects helped me improve my coding skills and gain practical experience in building real-world applications using C.

### 1. Basic Calculator

A console-based calculator that performs basic arithmetic operations.

**Features**

* Subtraction
* Multiplication
* Division

* User-friendly menu-driven interface

**Concepts Used**

* Functions
* Switch-case
* Arithmetic Operators
* Conditional Statements

### 2. Matrix Operations

A collection of C programs that perform common matrix operations and demonstrate the use of arrays, loops, and mathematical logic.

**Features**

* Matrix Addition
* Matrix Multiplication
* Matrix Transpose

**Concepts Used**

* Two-Dimensional Arrays
* Nested Loops

### 3. Student Management System

A menu-driven application that manages student records using structures and file handling for permanent data storage.

**Features**

* Add Student Record
* Display All Student Records
* Search Student by Roll Number
* Update Student Details
* Delete Student Record

**Concepts Used**

* Structures
* Functions
* Binary File Handling
* Menu-Driven Programming

### 4. Bank Account Management System

**Features**

* Deposit Money

**Concepts Used**

* Structures
* Functions
* File Handling

* Menu-Driven Programming
